## Usage

This project is made possible with:
- Node.js > lts/gallium
- Next.js
- React.js
- styled-components + tailwind.css
- Uncountable creative design effort

To start with the repository, run:
```sh
> nvm install
> nvm use
> yarn
```

To start developing: 
```sh
> yarn dev
```
